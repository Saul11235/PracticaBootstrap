Ejemplos de uso de grid - Bootstrap

En este caso se tomaran en cuenta dos nuevos elementos que podemos usar en bootstrap, las clases row y column respectivamente Recuerda que los elementos col deben estar dentro de row, para estos ejemplos.

  • .container se ha coloreado de Morado
  • .row se ha coloreado de Azul-Celeste
  • .col se ha coloreado de Rojo-Naranja
