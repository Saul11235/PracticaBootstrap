Ejemplos de uso de grid

En este caso se tomaran en cuenta dos nuevos elementos que podemos usar en bootstrap, las clases row y column respectivamente Recuerda que los elementos col deben estar dentro de row, para estos ejemplos.

  • .container se ha coloreado de Morado
  • .row se ha coloreado de Azul-Celeste
  • .col se ha coloreado de Rojo-Naranja

Ejemplo #3

En bootstrap podemos personalizar los anchos de las columnas asignando por ejemplo pesos de ponderación tomando como maximo 12, es decir si tomamos un peso 6 estaremos asignando la mitad si tomamos 4 estaremos tomando la tercer parte etc.
En este caso tomaremos diferentes pesos dentro del container hay dos elementos .row en el primero se ha colocado 2 elementos .col en el primero y 3 en el segundo. Bootstrap dividirá la fila en un numero equivalente de columnas.

Nota: el estilo fondo naranja no se aplica a las columnas del medio porque no son clases .col sino son clases .col-5 y .col-6 respectivamente.
